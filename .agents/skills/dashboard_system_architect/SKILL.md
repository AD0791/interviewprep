# Dashboard and System Architect Skill

This skill focuses on building responsive, high-performance dashboards, real-time monitoring interfaces, and integrating them with aggregated backend APIs.

## Core Responsibilities
1. Design layouts for dashboards (widgets, responsive grid systems, Tailwind/ChakraUI integrations).
2. Optimize React rendering performance (preventing unnecessary re-renders, windowing/virtualization for long data tables, debouncing chart updates).
3. Architect state management and local caches for metric data (using Redux Toolkit slices and SWR/React Query caching policies).
4. Establish clear API response contracts for charts (formatting timeseries data, categorical distributions, and key performance indicators (KPIs)).
5. Guide data transformations (mapping nested JSON payloads into flat structures required by charting libraries).
