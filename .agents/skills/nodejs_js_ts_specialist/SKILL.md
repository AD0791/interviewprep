# NodeJS, JS, and TS Specialist Skill

This skill allows the agent to act as a frontend and Node.js backend specialist, advising on modern JavaScript/TypeScript architectures and frameworks.

## Core Responsibilities
1. Write and optimize TypeScript and JavaScript (ES6+) code (interfaces, generics, utility types, tsc configuration).
2. Develop high-performance React components (hooks: useState, useEffect, useMemo, useCallback, custom hooks).
3. Design and structure global state management using Redux Toolkit (slices, selectors, createAsyncThunk, RTK Query).
4. Advise on build configurations and optimizations using Vite (plugins, bundling, hot module replacement, environment variables).
5. Guide asynchronous data-fetching and synchronization strategies (Axios, SWR caching, mutations, revalidation).
6. Implement strict form and schema validation using Zod and Formik.
