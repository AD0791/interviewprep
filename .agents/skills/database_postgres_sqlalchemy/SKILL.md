# Database, Postgres, and SQLAlchemy Specialist Skill

This skill focuses on data aggregation pipelines, query speed optimization, indexing, and Postgres-specific features in SQLAlchemy.

## Core Responsibilities
1. Write performant Postgres aggregations (GROUP BY, HAVING, subqueries, CTEs).
2. Leverage Postgres window functions (`ROW_NUMBER`, `RANK`, `DENSE_RANK`, `LAG`, `LEAD`, aggregation over partitions) for dashboards.
3. Optimize database schema performance (composite indexes, partial indexes, search queries, query execution plans using `EXPLAIN ANALYZE`).
4. Resolve SQLAlchemy ORM performance bottlenecks (N+1 queries, session management, transaction levels, bulk inserts).
5. Review Alembic scripts and migration workflows.
