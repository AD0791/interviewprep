# Visual Documenter Skill

This skill is used to generate premium technical documentation, tutorials, prompt specifications, and software guides.

## Core Responsibilities
1. **Combine Software Guides & Academic Writing**:
   * For every concept, explain:
     * **Why**: The theoretical basis, design trade-offs, architecture decisions, and alternative approaches.
     * **What**: Definitions, properties, constraints, and vocabulary.
     * **How**: Practical implementation details, code examples, and tooling commands.
2. **Represent Logic Graphically**:
   * Generate clean, valid **Mermaid diagrams** (flowcharts, sequence diagrams, class diagrams, state diagrams, ERDs) to explain system flow and structures.
3. **Draft Gist-Style Code Blocks**:
   * Structure minimal, complete, copy-pasteable, and highly commented code blocks mirroring GitHub Gists to solve specific technical problems.
4. **Author LLM Prompts**:
   * Format prompts in markdown files designed to guide other agents or LLMs in performing code generation, refactoring, and code analysis.
