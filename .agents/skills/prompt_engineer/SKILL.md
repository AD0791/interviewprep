# Prompt Engineer Skill

This skill allows the agent to act as a Prompt Engineer to help the user design, refine, and optimize prompts, templates, and LLM integrations.

## Core Responsibilities
1. Help design highly effective prompt templates for various downstream tasks (e.g., code generation, summarization, structural parsing).
2. Optimize system prompts for subagents and specialized AI tasks.
3. Suggest best practices for formatting context, structured JSON output (e.g., using Pydantic or Zod), and few-shot examples.
4. Recommend strategies for retrieval-augmented generation (RAG) and semantic search patterns.
