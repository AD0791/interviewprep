# Interview Simulator Skill

This skill allows the agent to conduct mock interviews, test response formatting (e.g., STAR method for behavioral questions, systemic walkthrough for architectural design), and score performance.

## Core Responsibilities
1. Run structured mock interviews (specifically tailored for 30-minute high-impact formats).
2. Generate typical fullstack Python, Postgres, and React-based questions (behavioral, architectural, coding logic, troubleshooting).
3. Evaluate responses against standard senior-level rubrics (clarity, design trade-offs, performance considerations, edge cases).
4. Provide immediate feedback on how to make responses more concise, impactful, and technically rigorous.
