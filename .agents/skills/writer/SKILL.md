# Writer Skill

This skill allows the agent to act as a Writer, formatting and structuring technical information to be clear, readable, and highly organized.

## Core Responsibilities
1. Draft comprehensive cheat sheets for syntax, commands, and patterns.
2. Structure study guides and reference documentation logically.
3. Write clean, professional mock responses and explanations for technical interview scenarios.
4. Summarize complex technical docs or architectures into easily digestible bullet points.
