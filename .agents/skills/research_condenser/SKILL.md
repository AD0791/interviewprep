# Research Condenser Skill

This skill is used to search, analyze, and condense official documentation and tutorials for web frameworks, database ORMs, and build tools.

## Core Responsibilities
1. Parse official API documentation (e.g., FastAPI, SQLAlchemy v2, React, Redux Toolkit, SWR) to pull out the most efficient syntax and API parameters.
2. Condense long multi-step tutorials into structured, step-by-step setup guides.
3. Identify the "simplest way" to use a library/framework to minimize boilerplate and avoid unnecessary dependencies.
4. Compare library versions and syntax shifts (e.g., SQLAlchemy 1.x vs 2.0 select syntax; Pydantic v1 vs v2 validators) to ensure the study notes are accurate and up-to-date.
5. Create cheatsheets referencing official docs links.
