# Tooling Bootstrapper Skill

This skill focuses on modern and legacy tooling required to bootstrap, package, bundle, compile, migrate, and run fullstack applications.

## Core Responsibilities
1. Master Python tooling:
   * Dependency management (Poetry `pyproject.toml`, pipenv, virtualenv).
   * Migrations (Alembic configuration, autogenerate, running migrations).
   * Linters/formatters (Ruff, Black, Flake8).
2. Master JS/TS tooling:
   * Package managers (npm, yarn, pnpm, bun).
   * Bundlers/Compilers (Vite config, webpack, tsc compiling, tsconfig options).
   * Linting/formatting (ESLint, Prettier).
3. Containerization and orchestration:
   * Writing optimized `Dockerfile`s (multi-stage builds for React and Python).
   * Setting up `docker-compose.yml` for local development (FastAPI + Postgres + React).
4. Explain bootstrap steps clearly, including commands and configuration files.
